Major Project: Smart Street Light Monitoring

Goal: Automate fault detection and reduce maintenance delays in urban lighting systems. View on GitHub

System Architecture

Uses ESP32 as the brain, reading LDR (Light) and Current sensors. Data is sent via WiFi to a centralized web dashboard.

Fault Detection Logic

Dual-sensor validation: If current is present but light is absent, a bulb failure is reported. If both are absent, a power failure is detected.

Web Dashboard

Real-time status display (Green for Working, Red for Fault) with event-driven updates to reduce network load.

Technical Specifications

  • Sensors: LDR (Brightness), Current Sensor (Power Flow)
  • Power: AC to DC via Bridge Rectifier
  • ADC: Raw ADC to Voltage calculation for accurate thresholds
  • Scalability: Ready for LoRa/GSM integration for long-range coverage
